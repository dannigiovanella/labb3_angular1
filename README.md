# Labb 3 - Angular 1

## Angular Konverteringsapp
Detta projekt är en webbapplikation utvecklad med Angular. Syftet med applikationen är att visa grundläggande förståelse för Angulars struktur, komponenter, routing och databindning.

## Länk till webbplats:

## Funktioner:
- Navigationsmeny med routing mellan undersidor
- Startsida med introduktion
- Konverteringssida för:
  - Meter och Foot
  - Celsius och Fahrenheit
- 404-sida för ogiltiga adresser
- Responsiv och enkel design

## Databinding
I applikationene används flera typer av databinding:

- Two-way binding (ngModel) - för att koppla inputfält till variabler
- Event binding (click) - för att trigga konverteringar
- Interpolation ({{ }}) - för att visa resultat

## Struktur
Applikationen är uppbyggd med komponenterna:

- App (Huvudkomponent)
- Home (startsida)
- Convert (konverteringssida)
- About (informationssida)
- NotFound (404-sida)
- Navbar (navigationsmeny)
